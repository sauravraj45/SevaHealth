# SevaHealth – Healthcare Web Application  

**Live Demo:** https://sevahealth.netlify.app

---

## Overview  

SevaHealth is a comprehensive healthcare web application designed to provide users with seamless access to a wide range of medical services and health-related information. The platform integrates multiple healthcare functionalities such as symptom analysis, online consultation, emergency support, and charitable services into a single, user-friendly interface.

It aims to bridge the gap between patients and healthcare services by enabling remote access and improving accessibility beyond geographical limitations.

---

## Features  

- 🧠 **Symptom Checker** – Analyze symptoms and get instant health insights  
- 👨‍⚕️ **Consultation** – Connect with healthcare professionals remotely  
- 💊 **Remedies** – Access basic health remedies and guidance  
- 🚑 **Emergency Support** – Quick access to emergency assistance  
- 🆘 **SOS Feature** – Immediate help in critical situations  
- 🔍 **Second Opinion** – Get additional medical opinions for better diagnosis  
- ❤️ **Charity Support** – Explore and contribute to healthcare-related initiatives  
- 🎥 **Video Calling** – Enable virtual doctor consultations  
- 📱 Fully responsive and user-friendly interface  

---

## Tech Stack  

- **Frontend:** React.js, Tailwind CSS  
- **Routing:** React Router  
- **Deployment:** Netlify  

---

## Project Objective  

The objective of this project is to develop a centralized healthcare platform that:  

- Provides multiple healthcare services in one place  
- Enables remote medical consultation and support  
- Enhances accessibility to healthcare resources  
- Delivers a modern and intuitive user experience

## Installation and Setup  

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/sauravraj45/SevaHealth.git

# Navigate to project directory
cd SevaHealth

# Install dependencies
npm install

# Start development server
npm run dev

```
